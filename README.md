## Предсказание оттока клиентов для провайдера интернет услуг и телефонии

Цель проекта: построить модель для прогноза оттока клиентов со значением метрики ROC-AUC на тестовой выборке не менее 0.85.

В рамках проекта были загружены и предобработаны данные, проведен EDA, построен пайплайн моделей, а также реализовано нейросетевое решение для предсказания оттока клиентов. По итогам исследования получены рекомендации для бизнеса, способствуюзие росту степени удержания клиентов.
